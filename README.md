# Sample D3.js application written in Kotlin

#### Install

```
git clone git@github.com:xfeldman/kotlin-d3-sample.git
cd kotlin-d3-sample
npm i
./gradlew build
http-server
```

#### Import to IDEA

File -> New -> Project From Existing Sources ...

_Select Gradle, then "Auto-sync project"_
